# Zombie Budget Design System

## Visual Theme
This is a PROFESSIONAL finance app with SPOOKY Halloween theming. Think premium fintech meets haunted house - clean, trustworthy, with subtle spooky elements.

## Color Palette (Inspired by FNBC Finance App + Halloween)
- **Background Gradient:** #0f0a1f → #1a1230 → #251d3a (deep purple gradient)
- **Brand Purple:** #8b5cf6 (primary actions, highlights)
- **Purple Light:** #a78bfa (hover states, accents)
- **Purple Dark:** #7c3aed (pressed states)
- **Accent Green:** #10b981 (positive actions, success)
- **Accent Red:** #ef4444 (negative actions, danger)
- **Accent Orange:** #f59e0b (warnings, alerts)
- **Neutral Grays:** #f1f5f9 to #0f172a (text, borders, cards)

## Design Principles
1. **Professional First:** Clean layouts, clear hierarchy, trustworthy aesthetics
2. **Spooky Second:** Subtle Halloween elements (ghosts, moons, zombies) as decorative accents
3. **Purple Dominance:** Purple is the primary brand color, not red/green
4. **Gradient Backgrounds:** Smooth purple gradients create depth without being distracting
5. **Glass Morphism:** Cards use backdrop-blur and semi-transparent backgrounds

## Typography
- **Headers:** Bold, clean sans-serif (Inter)
- **Body:** Regular weight, excellent readability
- **Numbers:** Monospace for financial data
- **No distressed fonts:** Keep it professional

## Animation Principles
- Smooth 200ms transitions for UI elements
- 60fps game animations using requestAnimationFrame
- Subtle float animations for decorative elements
- Scale transforms on hover (1.05x max)
- Glow effects on primary CTAs

## Component Patterns
- **Cards:** Semi-transparent background, border with purple tint, backdrop blur
- **Buttons:** Solid purple primary, outlined secondary, generous padding
- **Hover States:** Scale up slightly, enhance shadows, brighten colors
- **Focus States:** Purple ring for accessibility
- **Loading States:** Simple zombie emoji with pulse animation

## Color Usage Guidelines
- **Purple:** Primary CTAs, brand elements, highlights
- **Green:** Positive transactions, success states, healing
- **Red:** Negative transactions, danger states, zombie attacks
- **Orange:** Warnings, alerts, medium priority
- **Neutral:** Text, borders, backgrounds

## Avoid
- ❌ Excessive red/green (too intrusive)
- ❌ Black/gray/purple muddy combinations
- ❌ Distressed or grunge textures
- ❌ Over-the-top Halloween decorations
- ❌ Clashing color combinations
